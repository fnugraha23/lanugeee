Cloud Computing Remote Sensing

LontaraGeo Engine:
Otomasi Analisis Spasial

Integrasi Google Earth Engine API untuk pemrosesan citra satelit Sentinel-2 dan Landsat secara real-time guna mendukung keputusan manajemen Sub-DAS Tanralili.

Buka Aplikasi Fullscreen Lihat Demo Di Sini

Metodologi & Inovasi

Proyek ini dikembangkan untuk memecahkan masalah efisiensi dalam pengolahan data spasial untuk tesis saya mengenai Kinerja Sub-DAS Tanralili. Daripada mengunduh data citra satelit secara manual, sistem ini memanfaatkan cloud-based processing.

Sistem melakukan Cloud Masking otomatis untuk memastikan data yang dianalisis bebas dari gangguan awan, kemudian menghitung indeks spektral seperti NDVI dan EVI untuk memantau kerapatan vegetasi di wilayah tangkapan air.

Fitur Unggulan

  • 01. Pemrosesan Tanpa Beban: Seluruh komputasi berat dilakukan di server Google, sehingga aplikasi tetap ringan di sisi pengguna.
  • 02. Dinamis AOI: Pengguna dapat menggambar area spesifik di peta menggunakan drawing tools untuk analisis lokal.
  • 03. Filter Temporal: Memungkinkan perbandingan kondisi tutupan lahan antar tahun secara instan.

Aplikasi WebGIS Interaktif

Gunakan kontrol di sisi kiri untuk mengatur parameter dan gambarlah area (poligon) pada peta untuk memicu proses Earth Engine.

💡 Tips: Gambar poligon di peta untuk memulai analisis
